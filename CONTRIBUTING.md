# Contributing to Axon

Thank you for your interest in contributing to Axon! We welcome contributions from everyone.

## Code of Conduct

By participating in this project, you agree to abide by our Code of Conduct (coming soon).

## How Can I Contribute?

### Reporting Bugs

- Use the GitHub issue tracker.
- Check if the bug has already been reported.
- Provide a clear, descriptive title and as much information as possible.

### Suggesting Enhancements

- Use the GitHub issue tracker.
- Describe the feature and why it would be useful.

### Pull Requests

1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. If you've changed APIs, update the documentation.
4. Ensure the build passes.
5. Issue that pull request!

## Development Setup

See the [README](README.md) for instructions on setting up your local development environment.

## Style Guide

- **Frontend**: Follow the existing React + TypeScript patterns. Use Tailwind CSS for styling.
- **Backend**: Use idiomatic Rust. Follow `rustfmt` guidelines.

## Questions?

Feel free to open an issue or contact the maintainers.
